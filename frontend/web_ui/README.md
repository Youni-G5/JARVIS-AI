# JARVIS Web UI

Interface web React pour interagir avec JARVIS.

## Installation
```bash
npm install
```

## Démarrage
```bash
npm start
```

Ouvre [http://localhost:3000](http://localhost:3000)

## Configuration
Créer `.env.local`:
```
REACT_APP_BRIDGE_API=http://localhost:8000
```

## Fonctionnalités
- Interface chat textuelle
- Enregistrement vocal (en cours)
- Affichage des actions JARVIS
- Thème sombre
- Responsive design
